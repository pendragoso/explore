Title: User Password Update Schema
Mutation details:
- userPasswordUpdate(email: String, currentPassword: String, newPassword: String): Result
  - email: String - Email address of the user whose password needs to be updated.
  - currentPassword: String - The current password of the user.
  - newPassword: String - The new password for the user.
  - Result - The result of the password update operation.
    - SUCCESS - Indicates that the password update was successful.
    - FAILURE - Indicates that the password update failed.Example:
```graphql
mutation UpdateUserPassword($email: String, $currentPassword: String, $newPassword: String) {
  userPasswordUpdate(email: $email, currentPassword: $currentPassword, newPassword: $newPassword) {
    result
  }
}
```
Variables:
```json
{
  "email": "john.doe@example.com",
  "currentPassword": "oldPassword123",
  "newPassword": "newPassword123"
}
```
