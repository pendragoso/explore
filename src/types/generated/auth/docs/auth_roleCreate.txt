Title: Role Management Schema
Mutation details:
- roleCreate(input: RoleCreateInput!): Role
  - input: RoleCreateInput! - Input object for creating a new role.
    - permissions: [String!]! - List of permissions for the role.
    - name: String! - Name of the role.
    - slug: String! - Unique slug for the role, no spaces allowed.
    - description: String - Description of the role.
  - Role
    - id: ID! - Unique identifier for the role.
    - createdAt: DateTime! - Date and time when the role was created.
    - createdBy: ID! - User ID who created the role.
    - description: String - Description of the role.
    - name: String! - Name of the role.
    - slug: String! - Unique slug for the role, no spaces allowed.
    - type: RoleType! - Type of the role (PUBLIC, INTERNAL, CUSTOM, SUBSCRIPTION, ORGANIZATIONAL).
    - permissions: [String!]! - List of permissions for the role.
    - organization: ID - Organization ID associated with the role.
    - updatedAt: DateTime! - Date and time when the role was last updated.
    - updatedBy: ID! - User ID who last updated the role.Example:
```graphql
mutation CreateRole($input: RoleCreateInput!) {
  roleCreate(input: $input) {
    id
    createdAt
    createdBy
    description
    name
    slug
    type
    permissions
    organization
    updatedAt
    updatedBy
  }
}
```
