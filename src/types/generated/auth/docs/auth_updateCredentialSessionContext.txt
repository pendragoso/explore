Title: Update Credential Session Context Schema
Mutation details:
- updateCredentialSessionContext(updateCredentialSessionInput: UpdateCredentialSessionInput!): CredentialSession!
  - updateCredentialSessionInput: UpdateCredentialSessionInput! - Input object for updating the credential session context.
    - organization: String - Organization ID to update the credential session context.
  - CredentialSession - Updated credential session object.
    - id: ID! - Unique identifier for the credential session.
    - createdAt: DateTime! - Date and time when the credential session was created. An RFC-3339 compliant DateTime Scalar.
    - validUntil: DateTime - Date and time when the credential session is valid until.
    - credential: ID! - Credential ID associated with the session.
    - organization: ID! - Organization ID associated with the session.
    - serviceToken: String - Service token associated with the session.
    - user: ID - User ID associated with the session.
    - store: Int - Store ID associated with the session.
    - permissions: [String!] - List of permission names associated with the session.
    - mode: Mode! - Mode of the session (LIVE or TEST).Example:
```graphql
mutation UpdateCredentialSessionContext($updateCredentialSessionInput: UpdateCredentialSessionInput!) {
  updateCredentialSessionContext(updateCredentialSessionInput: $updateCredentialSessionInput) {
    id
    createdAt
    validUntil
    credential
    organization
    serviceToken
    user
    store
    permissions
    mode
  }
}
```
