Title: Credential Service Token Management Schema
Mutation details:
- credentialServiceTokenCreate(input: CredentialServiceTokenCreateInput!): [CredentialServiceToken!]!
  - input: CredentialServiceTokenCreateInput! - Input object for creating a credential service token.
    - storeId: Int! - Store ID for the credential service token.
  - [CredentialServiceToken!]! - List of created credential service tokens.
    - id: ID - Unique identifier for the credential service token.
    - credential: Credential - Credential associated with the service token.
    - organization: String - Organization associated with the service token.
    - mode: Mode - Mode of the service token (LIVE or TEST).
    - storeId: Int - Store ID associated with the service token.
    - serviceToken: String - Service token string.Example:
