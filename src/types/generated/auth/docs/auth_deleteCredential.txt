Title: Delete Credential Mutation Schema
Mutation details:
- deleteCredential(id: ID!): Result
  - id: ID! - Unique identifier of the credential to be deleted.
  - Result - Enum representing the result of the delete operation.
    - SUCCESS - Indicates the credential was successfully deleted.
    - FAILURE - Indicates the credential deletion failed.Example:
```graphql
mutation DeleteCredential($id: ID!) {
  deleteCredential(id: $id)
}
```
Variables:
```json
{
  "id": "123"
}
```
